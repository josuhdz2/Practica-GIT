Este es un proyecto simple y que realmente no tiene demaciada utilidad mas que la de aplicar lo aprendido durante el curso de GIT
Este proyecto solo sera una pagina de login en la con un correo y constrasenia especificas se puede acceder
La pagina funciona asi: se obtienen los datos ingresados por el usuario, se comparan y se la comparacion es verdadera
se levanta una bandera en localstorage para manejar la sesion.
La ruta a home.html tambien esta protegida ya que llama a un script que verifica la sesion.


Se puede encontrar hosteada en : https://josuhdz2.github.io/Practica-GIT/
